Description

Griff is an experiemental project aimed at producing a personal knowledge system. The basic idea of a personal knowledge system is that facts in the form of a colored triadic relation between text strings (called the data graph) are entered via forms and a special text language. The knowledge can then be explored using formal concept analysis to structure query results.

The data graph is colored to distinguish between anonymous nodes and named nodes. The named nodes are accessible via their names (which are text strings). The anonymous nodes have no name and so are only accessible via queries containing variables.

Each query is simply a colored triadic relation. The coloring of nodes designates them as either variables or constants. Constants match to nodes in the data graph while variables range over nodes in the data graph when a query result is produced.
